# [《我们不是不爱了，是学会了生活》（现实成长）](https://leiantuo.github.io/)

恋爱的时候，我们总以为爱可以解决一切。

后来才发现，爱解决不了房租、通勤、加班、情绪和疲惫。

你们也曾经无话不谈，从凌晨聊到天亮；也曾因为一句晚安没说好就难过半天。可慢慢地，聊天记录变短，见面次数变少，不是因为不在乎，而是生活挤进来了。

成年人的世界很吵。

老板的信息、客户的电话、家庭的责任、未来的焦虑，每一样都比“你为什么没回我”更紧急。你们开始学会把情绪收起来，把浪漫延后，把拥抱留到周末。

有一天你忽然意识到——

原来成熟的爱情，不是天天黏在一起，而是各自奔跑，却默认方向一致。

你忙的时候，我不打扰
我累的时候，你不指责
我们偶尔争吵，但不怀疑彼此

爱情从热烈变成稳定，从心跳变成心安。

不是不爱了，是学会了生活；不是没有激情，而是明白陪伴比激情更久。

真正走到最后的两个人，从来不是最甜的，而是最能一起面对现实的。
